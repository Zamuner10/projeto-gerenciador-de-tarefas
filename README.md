# Gerenciador de Tarefas em Python

Este é um projeto inicial de **gerenciador de tarefas** desenvolvido em Python.  
Permite adicionar, mostrar e apagar tarefas através de um menu interativo no terminal.

## Funcionalidades

- Adicionar tarefas em quantidade
- Mostrar todas as tarefas
- Apagar tarefas pelo nome
- Menu interativo com validação de inputs
- Limpeza de tela para melhor visualização

## Tecnologias

- Python 3
- Terminal/Prompt de Comando

## Como usar

1. Clone o repositório:
git clone https://github.com/Zamuner10/projeto-gerenciador-de-tarefas.git

2. Entre na pasta:
cd portifolio

3. Execute o programa:
python gerenciadorTarefas.py

4. Siga o menu para adicionar, mostrar ou apagar tarefas.

## Observações

- Este projeto é um exercício para aprender Python e lógica de programação.
- Não utiliza banco de dados ou arquivos externos; todas as tarefas são temporárias na execução.
